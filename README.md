# 📱 متجر التطبيقات | App Store

## 🌟 نظرة عامة

متجر إلكتروني كامل لبيع وتوزيع التطبيقات، مبني بـ HTML/CSS/JavaScript نقي - يعمل في أي متصفح بدون خادم.

## ✨ المميزات

### 🎨 التصميم
- ✅ تصميم عصري ملون (Material Design 3)
- ✅ دعم كامل للعربية (RTL) والإنجليزية (LTR)
- ✅ زر تبديل اللغة الفوري
- ✅ متجاوب 100% (موبايل + تابلت + ديسكتوب)

### 📦 المحتوى
- ✅ **100 تطبيق** جاهز في 12 فئة
- ✅ تطبيقات أندرويد، ويب، قوالب، أدوات، ألعاب، وأكثر
- ✅ كل تطبيق له: أيقونة، وصف، تقييم، تحميلات، إصدار، حجم، مطور

### 🛒 التسوق
- ✅ سلة تسوق كاملة مع localStorage
- ✅ نظام كوبونات خصم (4 كوبونات جاهزة)
- ✅ إتمام الشراء (Checkout)
- ✅ مكتبة التطبيقات المشتراة

### 👤 حسابات المستخدمين
- ✅ تسجيل دخول وإنشاء حساب
- ✅ حساب أدمن (admin@appstore.com / admin123)
- ✅ حساب مستخدم (user@example.com / user123)
- ✅ صفحة "مكتبتي" للتطبيقات المشتراة

### ⚙️ لوحة تحكم الأدمن
- ✅ إحصائيات شاملة (تطبيقات، مبيعات، إيرادات، تحميلات)
- ✅ عرض أحدث التطبيقات

### 🔍 البحث والتصفية
- ✅ بحث فوري في الاسم والوصف والمطور والوسوم
- ✅ تصفية حسب 12 فئة
- ✅ ترتيب: الأكثر تحميلاً، الأعلى تقييماً، السعر، الأحدث

### ⭐ التقييمات
- ✅ نظام تقييم بالنجوم لكل تطبيق
- ✅ مراجعات المستخدمين بالعربية والإنجليزية

## 🚀 طريقة الاستخدام

1. **افتح `index.html`** في أي متصفح
2. **تصفح التطبيقات** أو ابحث عنها
3. **أضف للسلة** التطبيقات التي تريد
4. **سجل دخول** بحساب تجريبي أو أنشئ حساباً
5. **أتمم الشراء** واستمتع بتحميل تطبيقاتك!

## 📁 بنية المشروع

```
app-store/
├── index.html          # الصفحة الرئيسية
├── css/
│   └── style.css       # التصميم الكامل
├── js/
│   └── app-store.js    # كل الوظائف
└── data/
    └── apps-data.js    # قاعدة بيانات 100 تطبيق + كوبونات + مراجعات + مستخدمون
```

## 🛠️ التخصيص

### إضافة تطبيق جديد:
عدّل `data/apps-data.js` وأضف:
```javascript
{ id: 101, name: 'My App', category: 'android', price: 9.99, rating: 4.5, downloads: 1000, size: '20 MB', version: '1.0.0', developer: 'Me', icon: '📱', desc_ar: 'وصف', desc_en: 'Description', featured: false, screenshots: 4, updated: '2026-06-26', tags: ['tag1'] }
```

### إضافة كوبون:
```javascript
'MYCODE': { discount: 30, type: 'percent', desc_ar: 'خصم 30%', desc_en: '30% off' }
```
